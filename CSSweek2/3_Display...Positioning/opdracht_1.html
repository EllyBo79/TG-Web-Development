Welke positioning values zijn er:
  • static
    • Deze position is niet beinvloed door : top, bottom, left en right. Niet op een specifieke manier gepositioneerd.By default zijn HTML elementen Static

  • relative
    • Deze kan je wel verschuiven met: top, bottom, left en right. Andere content zal niet verschuiven, doordat je dit gebruikt.

  • fixed
    • Hier blijft je content op een vaste plek. Deze beweegt dus niet , ook als je scrollt. Je kan wel weer met: top, bottom, left en right bepalen waar dit komt te staan.

  • absolute
    • Deze is relative gepositioneerd tot het voorgaande element.
  • sticky
    • Deze wordt bepaalt aan de hand van je scroll gedrag. Je kan dus bepalen waar een bericht blijft hangen tijdens het scrollen.
  • Overlapping elements
    • Hier kunnen met behulp van de z-index de elementen overlappen. Je kan negeatief of positief stapelen.
