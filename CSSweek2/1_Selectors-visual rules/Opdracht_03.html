Er zijn 3 verschillende selectors:

  • Type selectors

    • Deze geeft exact aan welk element aangesproken wordt, zoals een H1. Voordeel is dat in dit geval bijvoorbeeld alle h1 die je in je document hebt, aan kunnen worden gepast.

  • Descendant selectors

    • The descendant selector matches all elements that are descendants of a specified element.Zoals: div p.In dit geval worden alle p elementen binnen de div aangesproken.Voordeel hiervan is dat je iets specifieker de p elementen aan kan spreken.

  • class selectors

    • Je kan een class aanmaken en als je deze aanspreekt, worden deze alleen aangepast. Voordeel is dat je echt heel specifiek 1 onderdeel kan aanpassen.
