Voorbeelden van pseudo elementen zijn:
  • ::first-line

    • Hiermee kan je de eerste regel van vb. een paragraaf aanpassen met de volgende eigenschappen: font properties color properties background properties word-spacing letter-spacing text-decoration vertical-align text-transform line-height clear

  • ::first-letter

    • Zie boven, maar dan geldt het voor de eerste letter.

  • ::before

    • Hiermee kan je vb een plaatje voor een tekst zetten

  • ::after

    • Hiermee kan je vb een plaatje achteraan de tekst zetten.

  • ::marker

    • Hiermee kan je lijst items aanpassen qua kleur en formaat.

  • ::backdrop

    • Dit is een element voor achter de achtergrond.

  • ::cue

    • Hiermee style je captions of andere cues. vb tekst over een video.

  • ::cue-region

    • zie boven, maar dan meer voor alle cues en niet enkele.

  • ::file-selector-button

    • Dit gaat over de input button of type="file" te stylen.

  • ::grammar-error

    • Als de user agent dit als grammaticaal incorrect beschouwd kan je dit aangeven door dit pseudo element te gebruiken.Door bijvoorbeeld rood te onderstrepen.

  • ::part

    • alle sub onderdelen met dezelfde attributen.

  • ::placeholder

    • De tekst in een placeholder of textarea element.

  • ::selection

    • Hiermee kan je een geselecteerd stuk anders stylen. Zoals kleur.

  • ::slotted()

    • Alle elementen die in een slot zijn geplaatst.

  • ::spelling-error

    • Bij verkeerd gespelde woorden.

  • ::target-text

    • Als je een specifiek stukje tekst zoekt en daar naartoe scrollt. Dan kan dat stukje tekst worden aangesproken.
