1. Het verschil tussen een mixin en inheritance is dat je de mixin compleet terug ziet in de code bij de class en bij inheritance valt dat weer in een nieuwe regel.
2. Hieronder een voorbeeld van het gebruik van media query in combinatie met variables.
$break-small: 320px; $break-large: 1200px; .profile-pic { float: left; width: 250px; @media screen and (max-width: $break-small) { width: 100px; float: none; } @media screen and (min-width: $break-large) { float: right; } }
