Verschillende soorten selectors

ID selector=deze selector kan je maar op 1 element toepassen.Deze moet altijd worden aangesproken met een #

Elly


Class selector=Deze selector kan je wel aan meerdere elementen geven. Handig als je meerdere elementen exact hetzelfde wilt stylen. Deze selector spreek je aan met een .(punt)

Bouma


Universal selector=Deze selector spreekt alles aan binnen een HTML bestand.

Group selectors= Als je meerdere selectors hebt met exact dezelfde styling, is het handiger om deze bij elkaar te zetten. Hierdoor heb je ook minder code nodig en ziet het er dus netter uit.
